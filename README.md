📊 India Ecommerce Sales Analysis Dashboard
🚀 Project Overview
This Power BI project provides a comprehensive analysis of sales performance and profitability for an ecommerce business operating in India. Developed this dashboard to demonstrate data cleaning, DAX modeling, and interactive visualization techniques.

📈 Key Business Metrics
Total Revenue: 438,000

Total Profit: 37,000

Key Categories: Clothing, Electronics, and Furniture

Geographic Focus: Sales distribution across Maharashtra, Delhi, Madhya Pradesh, and other Indian states.

🛠️ Technical Features
Data Cleaning: Handled raw dataset processing using Power Query.

Time Intelligence: Monthly sales trend analysis (January – December).

Interactive Visuals: Geographic Bubble Map and Category-wise Profit Bar Charts.
